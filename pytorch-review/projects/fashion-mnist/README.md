# FashionMNIST 图像分类项目

## 项目简介

这是你的第一个完整的 PyTorch 实战项目！我们将构建一个神经网络来识别 10 种不同的服装类别。

## 数据集介绍

**FashionMNIST** 是一个经典的图像分类数据集：
- **图像数量**：70,000 张（60,000 训练 + 10,000 测试）
- **图像大小**：28×28 像素（灰度图）
- **类别数量**：10 类

### 10 个类别

| 标签 | 类别（英文） | 类别（中文） |
|------|-------------|-------------|
| 0 | T-shirt/top | T恤 |
| 1 | Trouser | 裤子 |
| 2 | Pullover | 套衫 |
| 3 | Dress | 连衣裙 |
| 4 | Coat | 外套 |
| 5 | Sandal | 凉鞋 |
| 6 | Shirt | 衬衫 |
| 7 | Sneaker | 运动鞋 |
| 8 | Bag | 包 |
| 9 | Ankle boot | 短靴 |

## 项目目标

通过这个项目，你将：
1. ✅ 加载和预处理真实数据集
2. ✅ 设计并训练一个神经网络
3. ✅ 评估模型性能
4. ✅ 保存和加载训练好的模型
5. ✅ 对新图像进行预测


## 开发流程

我们将按照以下步骤完成项目：

### 第1步：数据加载（data_loader.py）
- 下载 FashionMNIST 数据集
- 创建 DataLoader
- 数据可视化

### 第2步：模型定义（model.py）
- 设计神经网络架构
- 定义前向传播

### 第3步：训练模型（train.py）
- 定义损失函数和优化器
- 实现训练循环
- 保存检查点

### 第4步：评估模型（evaluate.py）
- 在测试集上评估
- 计算准确率
- 可视化结果

### 第5步：预测（predict.py）
- 加载训练好的模型
- 对单张图像进行预测

## 预期效果

完成项目后，你的模型应该能达到：
- **训练准确率**：85-90%
- **测试准确率**：80-85%
