# CIFAR-10 图像识别项目

## 项目简介

这是你的第二个 PyTorch 实战项目！我们将构建一个卷积神经网络（CNN）来识别 10 种不同的物体。

## 数据集介绍

**CIFAR-10** 是一个经典的计算机视觉数据集：
- **图像数量**：60,000 张（50,000 训练 + 10,000 测试）
- **图像大小**：32×32 像素（彩色图，3通道 RGB）
- **类别数量**：10 类

### 10 个类别

| 标签 | 类别（英文） | 类别（中文） |
|------|-------------|-------------|
| 0 | Airplane | 飞机 |
| 1 | Automobile | 汽车 |
| 2 | Bird | 鸟 |
| 3 | Cat | 猫 |
| 4 | Deer | 鹿 |
| 5 | Dog | 狗 |
| 6 | Frog | 青蛙 |
| 7 | Horse | 马 |
| 8 | Ship | 船 |
| 9 | Truck | 卡车 |

## 与 FashionMNIST 的区别

| 特性 | FashionMNIST | CIFAR-10 |
|------|-------------|----------|
| **图像类型** | 灰度图 | 彩色图（RGB） |
| **图像大小** | 28×28 | 32×32 |
| **通道数** | 1 | 3 |
| **模型类型** | 全连接网络 | 卷积神经网络（CNN） |
| **难度** | ⭐⭐ | ⭐⭐⭐⭐ |

## 项目目标

通过这个项目，你将：
1. ✅ 学习处理彩色图像（RGB 3通道）
2. ✅ 掌握卷积神经网络（CNN）
3. ✅ 理解卷积层、池化层的作用
4. ✅ 实现更复杂的模型架构
5. ✅ 达到 70%+ 的测试准确率

## 新知识点

### 1. 卷积神经网络（CNN）
- 卷积层（Conv2d）：提取图像特征
- 池化层（MaxPool2d）：降低维度
- 批归一化（BatchNorm2d）：加速训练

### 2. 数据增强
- 随机裁剪
- 随机水平翻转
- 颜色抖动

### 3. 学习率调度
- 动态调整学习率
- 提升模型性能
